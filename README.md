# AutoLaunder Mod

A mod for Schedule I that allows you to launder money more easily

## Description

This mod makes laundering money more efficient and less time consuming by automatically restarting laundering operations at capacity using cash from storage racks and safes.

## Installation

1. Make sure you have [MelonLoader](https://melonwiki.xyz/#/) installed for Schedule I
2. Download the latest release of Auto Launder
3. Place the `AutoLaunder.dll` file in your Schedule I Mods folder
4. Launch the game

## Features

- Automatically restarts laundering operations using safes and storage racks in businesses as cash souce

## Requirements

- Schedule I
- MelonLoader
